******************************Como usar Emmet***************************
escribe la etiqueta o estructura y oprime ctrl+E para autocompletar
* escribe . y presiona tab y crea un div con una class
* .div>p>a ctrl+e crea las etiquetas con las internas
* escribe div*3 ctrl+e crea 3 div
* escribe ul>li*6>a ctrl+e para crear la estructura basica de un menu
* escribe ! y presiona Tab crea estructura inicial de html (Guardar el archivo como html)
* escribe div y presiona Tab completa el div o div*5 y tab
